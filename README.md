CODTECH AI Internship Tasks SummaryHere is a simple summary of the four mandatory tasks you must complete and submit before your internship deadline. All code must be saved in a GitHub repository with proper comments.  The Four TasksTask 1: Text Summarization ToolGoal: Create a Python script that uses Natural Language Processing (NLP) to shorten long articles into concise summaries.  What to turn in: A Python script showing the original text and its summary.  Task 2: Speech Recognition SystemGoal: Build a basic system that converts spoken audio into written text using pre-trained models or libraries.  What to turn in: A working system that can transcribe short audio clips.  Task 3: Neural Style TransferGoal: Implement an AI model that takes the artistic style of a painting and applies it to a regular photograph.  What to turn in: A Python script or Jupyter Notebook showing examples of the styled images.  Task 4: Generative Text ModelGoal: Create an AI text generator (using tools like GPT or LSTM) that can write full, readable paragraphs based on a prompt you give it.  What to turn in: A Jupyter Notebook showing the model generating text from user prompts.
